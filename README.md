# Quotes Recommendation Chatbot using NLP

This project is a chatbot that recommends motivational, love, and funny quotes based on user input.

## Technologies Used
Python, NLTK, Flask, HTML, CSS

## Features
- Motivation Quotes
- Love Quotes
- Funny Quotes
- Web-based Chatbot

## How to Run

1. Install libraries

pip install flask nltk

2. Run server

python app.py

3. Open chatbot

Open index.html
